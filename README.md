# CRUD-App
CRUD operations using MERN Stack
## TECH STACK
- React JS
- Next JS
- Express Js
- Node Js
- Mongo DB
- Axios
- Redux-Toolkit
- Mongoose
- CORS
## SETUP & TESTING
Clone/download this repository and run yarn install or npm install.<br>
After all the dependencies are downloaded, run yarn start or npm start to run the app.
